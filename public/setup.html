Let's get your Ring Camera Server configured!

1Telegram Bot Setup

Create your Telegram bot:

  1. Open Telegram and message @BotFather
  2. Send /newbot
  3. Choose a name for your bot (e.g., "My Ring Camera Bot")
  4. Choose a username (e.g., "myringcamerabot")
  5. Copy the bot token from BotFather's response

Get Your Telegram User ID:

  1. Message @userinfobot on Telegram
  2. Copy your User ID from the response

2Ring Account Setup

Get your Ring refresh token:

  1. Click "Get Ring Token" below
  2. Enter your Ring email and password
  3. Complete 2FA if prompted
  4. Copy the refresh token

3OpenAI API Setup

Get your OpenAI API key:

  1. Go to OpenAI API Keys
  2. Sign in or create an account
  3. Click "Create new secret key"
  4. Copy the API key (starts with "sk-")

4Complete Setup

Click the button below to save your configuration and start the bot!
